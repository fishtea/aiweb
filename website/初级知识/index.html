📚 初级知识 ​

从零开始学习 AI 的基础知识体系。

本栏目适合第一次系统学习 AI 的读者。建议先建立“AI 能做什么、机器学习如何从数据中学习、深度学习为什么有效、大语言模型如何生成文本”的整体框架，再补齐数学、Python、数据、评估、NLP、CV、提示词和安全隐私等基础能力，最后进入进阶栏目学习 RAG、Agent、微调和评估。

主题简介
AI 学习路线图30 天入门计划、学习顺序、项目选择和检查点
人工智能入门AI 定义、发展历史、核心技术领域与未来趋势
数学基础线性代数、概率统计、微积分、优化和信息论的最小必要知识
Python 与数据处理基础Python、NumPy、Pandas、可视化和机器学习代码骨架
数据与特征工程数据类型、标签、特征工程、数据泄漏和数据质量检查
机器学习基础监督学习、无监督学习、回归/分类/聚类等核心算法
模型训练与评估基础训练/验证/测试、过拟合、评估指标、基线模型和上线检查
深度学习入门神经网络原理、反向传播、CNN/RNN 与深度学习框架
自然语言处理基础文本分类、信息抽取、token、Embedding 和 NLP 任务全景
计算机视觉基础图像分类、检测、分割、OCR、CNN 和视觉评估指标
生成式 AI 基础文本、图像、代码、音视频生成的基本机制、参数和风险
大语言模型基础Transformer 架构、GPT 系列、LLM 工作原理与应用
提示词入门任务、背景、约束、格式、示例和提示词常见误区
AI 伦理安全与隐私幻觉、偏见、隐私、提示词注入、人在回路和安全检查

💡 建议顺序：AI 学习路线图 → 人工智能入门 → 数学基础 → Python 与数据处理基础 → 数据与特征工程 → 机器学习基础 → 模型训练与评估基础 → 深度学习入门 → 自然语言处理基础/计算机视觉基础 → 生成式 AI 基础 → 大语言模型基础 → 提示词入门 → AI 伦理安全与隐私

全部内容基于真实在线资源总结编写

学完后可以继续 ​

  • 想做 AI 应用：进入 提示词工程 和 RAG 检索增强。
  • 想了解主流模型：进入 模型专区 对比 GPT、Claude、LLaMA、DeepSeek、Gemini、Qwen 等模型。
  • 想按知识库方法学习：阅读 知识库使用指南。

如何使用本栏目 ​

  • 零基础读者按表格顺序阅读，不建议跳过数据、评估和安全章节。
  • 产品、运营和管理者可重点阅读人工智能入门、生成式 AI、大语言模型、提示词和伦理安全。
  • 开发者应补齐 Python、数据与特征工程、模型训练评估，再进入进阶学习。
  • 每篇专题末尾的“精选资源”由采集脚本维护，正文内容优先按学习逻辑人工整理。
