# 进阶学习 — 迈向实践应用

> 你已经掌握了 AI 的基础知识，现在是时候深入实践了。进阶学习阶段将带你从理论走向真实世界的 AI 应用开发。

## 📋 前置要求

在进入本阶段之前，请确保你已经完成：

- ✅ **基础知识**：理解机器学习基本概念、神经网络原理
- ✅ **编程基础**：熟练使用 Python，了解 PyTorch/TensorFlow 基本操作
- ✅ **LLM 基础**：了解 Transformer 架构、自注意力机制、预训练与微调的基本概念
- ✅ **工具熟悉**：能够使用命令行、Git、Jupyter Notebook

## 🎯 你将学到什么

进阶学习阶段涵盖以下核心主题，建议按顺序学习：

### 1. 模型微调技术
学习如何将预训练模型适配到特定任务和领域。从数据准备到训练配置，从全量微调到参数高效微调，你将掌握让开源模型为你所用的完整技能。

### 2. RAG 检索增强生成
掌握让 LLM 连接外部知识库的核心技术。学习文档分块、向量存储、检索策略，构建知识密集型应用的黄金方案。

### 3. Agent 智能体
探索 AI 自主推理和行动的能力。从工具使用到多步规划，从单智能体到多智能体协作，解锁 LLM 执行复杂任务的潜力。

### 4. 提示词工程
系统学习 Prompt 设计方法论。从基础模式到高级技巧，掌握与 AI 模型高效沟通的艺术。

### 5. 模型评估与基准
学习如何科学地衡量模型能力。从标准化基准到自定义评估，建立量化比较的思维框架。

## 🗺️ 建议学习路径

```
模型微调技术 → RAG检索增强 → Agent智能体
         ↓                          ↓
   提示词工程 ←————————— 模型评估与基准
```

- **第一步**：学习**模型微调技术**，理解如何定制模型
- **第二步**：学习**RAG 检索增强**，掌握知识注入方法
- **第三步**：学习 **Agent 智能体**，解锁自主推理能力
- 在学习过程中持续练习**提示词工程**，提升与模型交互的效率
- 最后通过**模型评估与基准**，建立量化评估的思维

## 💡 学习建议

1. **动手实践**：每个主题都配有实践指导，请务必动手操作
2. **循序渐进**：按推荐顺序学习，前面的内容是后面的基础
3. **组合应用**：进阶技术的真正威力在于组合使用——例如用微调后的模型作为 Agent 的基础，结合 RAG 提供知识支持
4. **记录笔记**：AI 领域发展迅速，建立个人知识库非常重要
