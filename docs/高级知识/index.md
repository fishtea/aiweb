# 🧠 高级知识

深入 AI 前沿领域，涵盖训练优化、多模态、安全与架构研究。

本栏目面向已经掌握 LLM 应用基础、希望理解模型底层能力来源和工程边界的读者。内容重点不是“追热点名词”，而是理解训练数据、模型结构、优化方法、安全约束和多模态能力如何影响真实系统。

| 主题 | 简介 |
|------|------|
| [模型训练与优化](模型训练与优化/index.md) | 分布式训练、量化、蒸馏、推理加速 |
| [AI 安全与对齐](AI安全与对齐/index.md) | 安全问题、对齐技术、RLHF、红队测试 |
| [多模态模型](多模态模型/index.md) | CLIP、LLaVA、视觉语言模型等 |
| [模型架构研究](模型架构研究/index.md) | MoE、Mamba、状态空间模型等前沿架构 |
| [数据工程与合成数据](数据工程与合成数据/index.md) | 训练、微调、RAG、评估中的数据治理与合成数据方法 |

> 💡 建议顺序：数据工程与合成数据 → 训练优化 → 多模态 → 架构研究 → 安全对齐

## 学习建议

- 如果目标是微调或评估，先理解数据清洗、样本分层、合成数据和版本管理。
- 如果目标是部署和推理优化，优先阅读模型训练与优化，再结合模型专区做选型。
- 如果目标是研究前沿模型能力，按多模态、架构研究、安全对齐的顺序扩展。

## 前置知识

- 已理解机器学习、深度学习和大语言模型的基本概念。
- 能读懂模型训练、推理、评估、RAG、Agent 等常见工程术语。
- 对 Python、PyTorch、Hugging Face 或云端模型 API 有基础实践经验。

## 学完后的能力

- 能判断模型训练、微调、量化、蒸馏和推理优化各自解决的问题。
- 能从数据质量、评估集、安全约束和部署成本角度分析一个 AI 系统。
- 能阅读模型技术报告，并区分架构创新、数据策略和工程优化的贡献。
