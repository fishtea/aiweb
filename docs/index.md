# 🤖 AI 学习路线图 — 从入门到精通

欢迎来到 **AI 学习资源库**。这是一份精心策划的 AI 学习路径，所有课程和资源均经过实际调研，引用真实、可访问的在线课程和文章。

## 🧭 学习路径总览

```
初级知识（6-12 个月）
├── 人工智能入门         → 了解 AI 是什么、历史与发展
├── 机器学习基础         → 掌握监督/无监督学习核心算法
├── 深度学习入门         → 学习神经网络与深度学习框架
└── 大语言模型基础       → 理解 LLM、Transformer 与 GPT
                ↓
中级进阶（3-6 个月）
├── NLP 与计算机视觉
├── 生成式 AI（GAN、扩散模型）
└── MLOps 与模型部署
                ↓
高级专题（持续学习）
├── Agentic AI 与多智能体系统
├── AI 安全与伦理
└── 前沿研究（Transformer 变体、多模态等）
```

## 🗺️ 三条学习路径（基于真实路线图）

根据 [Krish Naik 的 2025 AI Mastery Roadmap](https://github.com/krishnaik06/Complete-RoadMap-To-Learn-AI)，以下是三条经典路径：

### 路径一：数据科学与经典 AI（约 4 个月）
**目标岗位：** 数据科学家、机器学习工程师  
**核心内容：** Python → 统计学 → 机器学习 → 深度学习 → NLP → 计算机视觉 → 部署/MLOps  
**推荐资源：** [Coursera ML Specialization](https://www.coursera.org/specializations/machine-learning-introduction)

### 路径二：生成式 AI（约 2 个月）
**目标岗位：** 生成式 AI 工程师、AI 产品开发者  
**核心内容：** 提示工程 → RAG → 微调 → LangChain → Hugging Face → 扩散模型  
**推荐资源：** [Deep Learning Specialization](https://www.coursera.org/specializations/deep-learning) + [LangChain 课程](https://www.udemy.com/course/develop-llm-powered-applications-with-langchain/)

### 路径三：Agentic AI（约 2 个月）
**目标岗位：** AI 架构师、AI Agent 开发者  
**核心内容：** LangGraph → AutoGen → CrewAI → 工具调用 → 多智能体系统  
**推荐资源：** 参见 [Krish Naik 的 Agentic AI 路线图](https://github.com/krishnaik06/Roadmap-To-Learn-Agentic-AI)

## 🎯 如何选择你的起点

| 你的背景 | 建议起点 | 预计时间 |
|---------|---------|---------|
| 完全零基础 | [AI For Everyone](https://www.coursera.org/learn/ai-for-everyone) | 6 小时 |
| 有编程基础（Python） | [ML Specialization](https://www.coursera.org/specializations/machine-learning-introduction) | 3 个月 |
| 想做 AI 产品/应用 | 先学 ML Specialization → 再学 Deep Learning Specialization | 6 个月 |
| 想深入研究 LLM | 学完 ML → DL → 读 [Lakera LLM Guide](https://www.lakera.ai/blog/large-language-models-guide) | 6-9 个月 |

## 🆓 免费资源推荐

1. **[Microsoft AI for Beginners](https://microsoft.github.io/AI-For-Beginners/)** — 12 周、24 课时的免费课程体系，覆盖 AI 历史、神经网络、计算机视觉、NLP、伦理等。使用 TensorFlow 和 PyTorch。
2. **[Andrew Ng 原始 ML 课程（Stanford CS229）](https://www.andrewng.org/courses/)** — 2011 年首次在线发布，吸引了超 10 万学生，是现代 MOOC 的起点。
3. **[DeepLearning.AI 短课程](https://www.deeplearning.ai/courses/)** — 30 分钟到 1 小时的免费短课程，覆盖 ChatGPT 提示工程、LangChain、RAG 等。
