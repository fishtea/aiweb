## 📖 四大学习板块

### 1️⃣ 初级知识 — 夯实基础

适合 **零基础学习者**，无需任何 AI 经验，只需具备基本的编程和数学概念。完成本阶段后，你将掌握 AI 的核心术语、原理和工作流程。

| 章节 | 内容 | 预计时间 |
|------|------|----------|
| [人工智能入门](初级知识/人工智能入门/) | 什么是 AI，历史与分类，应用场景 | 1-2 天 |
| [机器学习基础](初级知识/机器学习基础/) | 监督/无监督/强化学习，经典算法 | 3-5 天 |
| [深度学习入门](初级知识/深度学习入门/) | 神经网络，CNN/RNN/Transformer，框架 | 3-5 天 |
| [大语言模型基础](初级知识/大语言模型基础/) | LLM 原理，Tokenization，Prompt，主流模型 | 2-3 天 |

### 2️⃣ 进阶学习 — 深入方向

在掌握基础知识后，选择感兴趣的方向深入学习。

- **自然语言处理（NLP）** — 文本分类、命名实体识别、情感分析
- **计算机视觉（CV）** — 图像分类、目标检测、图像生成
- **提示工程（Prompt Engineering）** — 结构化提示、思维链、Few-shot
- **检索增强生成（RAG）** — 知识库问答、向量检索、文档理解

### 3️⃣ 高级知识 — 专家之路

深入 AI 的核心前沿技术。

- **强化学习** — Q-Learning、PPO、RLHF
- **多模态模型** — 图文理解、视频理解、语音处理
- **AI Agent** — 工具调用、自主规划、多 Agent 协作
- **模型微调** — LoRA、QLoRA、全参数微调
- **LLMOps** — 模型部署、推理优化、监控评估

### 4️⃣ 🛠 模型与工具概览

了解和选择适合你的工具：

| 类别 | 代表性工具/模型 |
|------|----------------|
| 闭源模型 | GPT-4o, Claude 3.5, Gemini 2.0 |
| 开源模型 | LLaMA 3, DeepSeek-V3, Qwen 2.5, Mistral |
| 深度学习框架 | PyTorch, TensorFlow, JAX |
| 训练工具 | Hugging Face Transformers, Axolotl, Unsloth |
| 推理部署 | vLLM, Ollama, llama.cpp, TGI |

## 💡 学习建议

1. **动手实践** — 每个概念都尝试用代码实现或使用在线工具体验
2. **做笔记** — 用自己的话复述学到的概念
3. **循序渐进** — 不要跳步，每个章节都建立在前一章基础上
4. **多问为什么** — 理解原理比记忆结论更重要
5. **加入社区** — 参与讨论、阅读论文、关注行业动态
