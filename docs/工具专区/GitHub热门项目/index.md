### 🥇 1. addyosmani/agent-skills

- **链接**: https://github.com/addyosmani/agent-skills
- **🌟 本周增长**: +11,088 (总 61,198)
- **一句话概括**: Google Chrome DevRel 维护的"技能"库，将工程工作流（规范驱动开发、TDD、可观测性）封装为 AI 编码 Agent 可重复使用的单元。
- **为什么值得关注**: 由 Google 高管 Addy Osmani 维护，定义了 Agent Skills 这一新范式——不只是给 Agent 工具，而是给 Agent"工程能力"。Shell 语言实现，MIT 许可，社区贡献活跃。
- **与本周热点的关联**: 与 NVIDIA SkillSpector 的发布相互呼应，"Skills" 生态正在成为 AI Agent 领域的基础设施层。

### 🥇 2. chopratejas/headroom

- **链接**: https://github.com/chopratejas/headroom
- **🌟 本周增长**: +10,660 (总 30,002)
- **一句话概括**: 一个"资深开发者" AI 编码 Agent，可以在代码仓库中自主导航、理解、修改和部署代码。
- **为什么值得关注**: Python 实现，持续多周保持高增长。定位介于 Cursor 和 Devin 之间——既不是 IDE 插件也不是全自动化工程师，而是"Senior Dev"级别的编码辅助。适合团队希望有 AI 同事协助 PR review、代码重构和单元测试编写的场景。

### 重点项目解读

#### 🥇 karpathy/autoresearch（89,803 ⭐）
Andrej Karpathy 的新项目，让 AI Agent 在**单个 GPU** 上自动执行 nanochat 训练研究。这代表了一个关键趋势：**研究自动化**——Agent 不再只是写代码，而是自主设计实验、训练模型、分析结果。100k Stars 在即，是 2026 年最具影响力的 AI 研究项目之一。

#### 🥇 HKUDS/nanobot（45,010 ⭐）
来自香港大学数据科学团队的轻量级 Agent 框架。强调**极简设计**：一个命令启动，LLM 自动发现并调用本地工具，支持 OpenAI/Anthropic 兼容 API。适合快速原型验证和个人工作流自动化。
