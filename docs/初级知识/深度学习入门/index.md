# 深度学习入门

> 神经网络如何工作？为什么深度学习能解决复杂问题？

---

## 什么是深度学习？

**深度学习（Deep Learning, DL）** 是机器学习的一个子集，它使用 **多层神经网络** 来学习数据的复杂模式和表示。

### 深度学习 vs 传统机器学习

| 对比维度 | 传统机器学习 | 深度学习 |
|----------|-------------|----------|
| **特征提取** | 需要人工设计特征 | 自动学习特征 |
| **数据量** | 少量数据即可 | 需要大量数据 |
| **计算资源** | CPU 即可运行 | 通常需要 GPU |
| **复杂任务** | 处理简单模式 | 处理复杂模式（图像、语音、文本） |
| **可解释性** | 较好理解 | 较难解释（"黑箱"） |

> 💡 **类比**：传统机器学习就像一个厨师按照菜谱做菜（人工特征）。深度学习就像一个学徒通过看成千上万道菜，自己学会了食材的组合规律（自动特征学习）。

## 神经网络基础

### 神经元（Neuron）—— 最小计算单元

神经网络的基本单元叫 **神经元**（也叫节点），灵感来源于生物神经元：

```
        输入 1 ─────┐
                    ├──→ (加权求和) → (激活函数) → 输出
        输入 2 ─────┘
```

一个神经元做了两件事：
1. **加权求和**——把输入乘以各自的权重后加总（加上偏置）
2. **激活**——将求和结果通过一个非线性函数（激活函数）

### 激活函数（Activation Function）

激活函数引入 **非线性**，让神经网络能够学习复杂的模式。常用激活函数：

| 函数 | 公式 | 特点 |
|------|------|------|
| **ReLU** | `f(x) = max(0, x)` | 最常用，计算快，缓解梯度消失 |
| **Sigmoid** | `f(x) = 1/(1+e⁻ˣ)` | 输出 0~1，用于二分类输出层 |
| **Tanh** | `f(x) = (eˣ-e⁻ˣ)/(eˣ+e⁻ˣ)` | 输出 -1~1，常用于 RNN |

### 网络层（Layer）

多个神经元组合成一层，多层堆叠形成深度网络：

```
输入层         隐藏层1         隐藏层2         输出层
  ○            ○   ○          ○   ○            ○
  ○     →      ○   ○    →     ○   ○     →      ○
  ○            ○   ○          ○   ○
  ○
```

- **输入层**：接收原始数据（如像素值）
- **隐藏层**：中间处理层，提取越来越抽象的特征
- **输出层**：输出最终结果（如分类概率）

> 💡 "深度"学习中的"深度"指的就是隐藏层的数量。现代大模型可能有几十甚至上百层。

## 常见网络架构

### 1️⃣ 卷积神经网络（CNN）—— 专为图像设计

**CNN（Convolutional Neural Network）** 通过卷积操作提取图像的局部特征。

- **核心思想**：用"滑动窗口"扫描图像，检测边缘、纹理、形状等特征
- **主要应用**：图像分类、目标检测、人脸识别
- **代表模型**：AlexNet, ResNet, YOLO

```
原始图像  →  [边缘检测]  →  [纹理检测]  →  [物体部件]  →  [完整物体]
  像素      低层特征        中层特征        高层特征       分类结果
```

> 💡 **类比**：CNN 的工作方式就像人类看画——先看线条和颜色（低层），再看形状（中层），最后识别出是"一只猫"（高层）。

### 2️⃣ 循环神经网络（RNN/LSTM）—— 专为序列设计

**RNN（Recurrent Neural Network）** 擅长处理序列数据（文本、语音、时间序列）。

- **核心思想**：网络有"记忆"，当前输出依赖于之前的输入
- **LSTM（长短期记忆网络）**：RNN 的改进版，解决了长期依赖问题（记住更早的信息）
- **主要应用**：机器翻译、语音识别、文本生成

> 💡 **类比**：RNN 就像读句子——每读一个词，都结合前面读过的内容来理解。LSTM 则像有更好记忆的人，能记住段落开头的关键信息。

### 3️⃣ Transformer —— 当今 AI 的基石

**Transformer** 是 2017 年 Google 提出的革命性架构，目前几乎所有主流 AI 模型都基于它。

- **核心创新**：**自注意力机制（Self-Attention）**——模型可以同时关注输入序列中的所有位置
- **优势**：可以并行计算（比 RNN 快得多），能处理长距离依赖
- **主要应用**：GPT、BERT、LLaMA、所有大语言模型
- **变体**：Vision Transformer（ViT）——把 Transformer 用在图像上

> 💡 **类比**：注意力机制就像你在读一句话时，会同时关注所有相关的词，而不是只按顺序一个一个看。

## 训练原理

### 前向传播（Forward Pass）

数据从输入层逐层向前传递，直到输出层产生预测结果。

```
输入 → 第1层 → 第2层 → ... → 输出层 → 预测结果
```

### 损失函数（Loss Function）

衡量模型的预测值与真实值之间的差距。

- **分类任务**：交叉熵损失（Cross-Entropy Loss）
- **回归任务**：均方误差（MSE）

> 损失越大，说明模型预测越不准。

### 反向传播（Backpropagation）

从输出层向输入层反向逐层计算每个权重的"责任"（梯度）。

```
损失 ← 梯度 ← 梯度 ← 梯度
```

### 梯度下降（Gradient Descent）

根据梯度更新权重，逐步减小损失：

```
新权重 = 旧权重 - 学习率 × 梯度
```

整个过程循环进行：

```
前向传播 → 计算损失 → 反向传播 → 更新权重 → 重复
```

| 概念 | 解释 | 类比 |
|------|------|------|
| **学习率** | 每次更新的步长大小 | 下山时迈的步子大小 |
| **Batch** | 一次训练用的样本数 | 一次看多少道题 |
| **Epoch** | 完整遍历一遍所有训练数据 | 做完一整套练习册 |
| **梯度消失** | 深层网络梯度太小无法更新 | 山底越来越平，走不动了 |

## 深度学习框架

### PyTorch（推荐）

- **特点**：动态计算图、调试方便、社区活跃
- **适用**：研究、原型开发、教学
- **简单易用**：`pip install torch`
- **代表项目**：Hugging Face Transformers、LLaMA、Stable Diffusion

### TensorFlow / Keras

- **特点**：工业部署完善、Keras API 易用
- **适用**：生产环境、移动端/Web 端部署
- **学习曲线**：略陡于 PyTorch

### 框架选型建议

| 场景 | 推荐 |
|------|------|
| 初学者学习 | PyTorch 或 Keras |
| 学术研究 | PyTorch |
| 工业部署 | TensorFlow 或 PyTorch |
| 移动/Web | TensorFlow Lite / ONNX |

## 实践心态

学习深度学习时，请记住：

1. **先理解直觉，再深入数学**——先知道"为什么这样做"，再理解"怎么算"
2. **从简单开始**——在 MNIST（手写数字识别）上跑通第一个神经网络
3. **多动手实验**——调整参数观察变化，是最好的学习方式
4. **利用预训练模型**——不必从头训练，Hugging Face 上有大量现成模型
5. **复现经典实验**——动手实现一个简单的 CNN 或 Transformer 组件

## ✅ 本章小结

- **深度学习** 使用多层神经网络自动学习特征
- **神经元** 是基本单元，进行加权求和 + 激活函数
- **CNN** 擅长图像，**RNN/LSTM** 擅长序列，**Transformer** 是当前主流
- 训练过程：前向传播 → 计算损失 → 反向传播 → 梯度下降更新
- **PyTorch** 是学习深度学习的首选框架
- 从简单项目开始，利用预训练模型，逐步深入
