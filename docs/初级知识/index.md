# 📚 初级知识

从零开始学习 AI 的基础知识体系。

本栏目适合第一次系统学习 AI 的读者。建议先建立“AI 能做什么、机器学习如何从数据中学习、深度学习为什么有效、大语言模型如何生成文本”的整体框架，再进入进阶栏目学习 RAG、Agent、微调和评估。

| 主题 | 简介 |
|------|------|
| [人工智能入门](人工智能入门/index.md) | AI 定义、发展历史、核心技术领域与未来趋势 |
| [机器学习基础](机器学习基础/index.md) | 监督学习、无监督学习、回归/分类/聚类等核心算法 |
| [深度学习入门](深度学习入门/index.md) | 神经网络原理、反向传播、CNN/RNN 与深度学习框架 |
| [大语言模型基础](大语言模型基础/index.md) | Transformer 架构、GPT 系列、LLM 工作原理与应用 |

> 💡 建议顺序：人工智能入门 → 机器学习基础 → 深度学习入门 → 大语言模型基础
> 
> *全部内容基于真实在线资源总结编写*
