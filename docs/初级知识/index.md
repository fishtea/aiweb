# 📚 初级知识

从零开始学习 AI 的基础知识体系。

本栏目适合第一次系统学习 AI 的读者。建议先建立“AI 能做什么、机器学习如何从数据中学习、深度学习为什么有效、大语言模型如何生成文本”的整体框架，再补齐数学、Python、数据、评估、NLP、CV、提示词和安全隐私等基础能力，最后进入进阶栏目学习 RAG、Agent、微调和评估。

## 学完后可以继续

- 想做 AI 应用：进入 [提示词工程](/进阶学习/提示词工程/) 和 [RAG 检索增强](/进阶学习/RAG检索增强/)。
- 想了解主流模型：进入 [模型专区](/模型专区/) 对比 GPT、Claude、LLaMA、DeepSeek、Gemini、Qwen 等模型。
- 想按知识库方法学习：阅读 [知识库使用指南](/知识库/)。

## 如何使用本栏目

- 零基础读者按表格顺序阅读，不建议跳过数据、评估和安全章节。
- 产品、运营和管理者可重点阅读人工智能入门、生成式 AI、大语言模型、提示词和伦理安全。
- 开发者应补齐 Python、数据与特征工程、模型训练评估，再进入进阶学习。
- 每篇专题末尾的“精选资源”由采集脚本维护，正文内容优先按学习逻辑人工整理。
