## 什么是人工智能？

**人工智能（Artificial Intelligence，简称 AI）** 是计算机科学的一个分支，旨在创建能够模拟人类智能的系统。简单来说，就是让机器具备"思考"和"学习"的能力。

想象一下：一个程序能够识别图片中的猫、听懂你的语音指令、或者跟你下棋并击败你——这些都属于人工智能的应用。

> 💡 **类比**：如果说普通程序是一本"说明书"——每一步都写好了怎么做；那 AI 程序更像是一个"学生"——它从例子中自己学习怎么做，而不是被一步一步地教。

## AI vs 机器学习 vs 深度学习 vs 数据科学

这四个概念经常被混用，但它们之间有明确的包含关系：

```
┌────────────────────────────────────────────┐
│              人工智能 (AI)                    │
│  ┌──────────────────────────────────────┐   │
│  │       机器学习 (ML)                   │   │
│  │  ┌──────────────────────────────┐    │   │
│  │  │    深度学习 (DL)             │    │   │
│  │  │  神经网络（含LLM）           │    │   │
│  │  └──────────────────────────────┘    │   │
│  └──────────────────────────────────────┘   │
│  ┌──────────────────────────────────────┐   │
│  │  数据科学 (Data Science)             │   │
│  │  (数据分析、统计、可视化)           │   │
│  └──────────────────────────────────────┘   │
└────────────────────────────────────────────┘
```

| 概念 | 定义 | 例子 |
|------|------|------|
| **人工智能 (AI)** | 让机器展现智能的广泛领域 | 象棋程序、语音助手 |
| **机器学习 (ML)** | AI 的子集，让机器从数据中学习 | 垃圾邮件过滤、推荐系统 |
| **深度学习 (DL)** | ML 的子集，使用多层神经网络 | 人脸识别、ChatGPT |
| **数据科学** | 从数据中提取知识的跨学科领域 | 用户行为分析、市场预测 |

## 人工智能简史

AI 的发展经历了多次高潮和低谷，了解这段历史有助于理解当前 AI 热潮的背景。

### 1950s — 诞生与梦想
- **1950年**：艾伦·图灵发表论文《计算机器与智能》，提出 **图灵测试** —— 如果一台机器能在对话中让人类无法分辨它是人还是机器，就说它具有智能
- **1956年**：**达特茅斯会议** —— 约翰·麦卡锡等人正式提出"人工智能"一词，标志着 AI 作为一门学科的诞生

### 1960s-1970s — 早期繁荣与第一次寒冬
- 早期成果：逻辑推理程序、简单对话系统（ELIZA）
- **AI 冬天**（1970s）：由于计算能力不足、技术局限，AI 研究资金大幅削减

### 1980s — 专家系统与第二次寒冬
- **专家系统**兴起：基于规则的 AI 系统在特定领域取得商业成功
- **第二次 AI 冬天**（1980s末）：专家系统维护成本高昂、难以扩展，研究再次遇冷

### 1990s-2010s — 机器学习时代
- **1997年**：IBM 的 **深蓝（Deep Blue）** 击败国际象棋世界冠军卡斯帕罗夫
- 统计学习方法崛起：支持向量机（SVM）、随机森林等算法成为主流

### 2010s — 深度学习革命
- **2012年**：AlexNet 在图像识别比赛中大幅领先，深度学习开始爆发
- **2016年**：Google DeepMind 的 **AlphaGo** 击败围棋世界冠军李世石
- **2017年**：Google 发表论文《Attention Is All You Need》，提出 **Transformer** 架构——这是当今所有大语言模型的基础

### 2020s — 大语言模型时代
- **2020年**：OpenAI 发布 **GPT-3**（1750亿参数），展示了惊人的文本生成能力
- **2022年**：**Stable Diffusion** 开源，引发 AI 图像生成热潮；**ChatGPT** 发布，两个月用户破亿
- **2023-2025年**：GPT-4、Claude、LLaMA、DeepSeek 等模型相继推出，AI 进入"百模大战"时代

## 人工智能的类型

### 狭义 AI（Narrow AI / Weak AI）
当前我们使用的所有 AI 都属于这一类。它们在 **特定任务** 上表现出色，但无法泛化到其他领域。

- ✅ 人脸识别系统（只能识别人脸，不能下棋）
- ✅ 语音助手（只能回答问题，不能开车）
- ✅ 推荐算法（只能推荐内容，不能写诗）

### 通用 AI（General AI / AGI）
指具备与人类相当的、通用的智能能力——能学习、推理、适应任何任务。**目前尚未实现**，是 AI 研究的终极目标。

> 💡 当前（2025年）我们处于"狭义 AI 高度发达，通用 AI 仍在探索"的阶段。像 ChatGPT 虽然能力强大，但它仍然没有真正的"理解"和"意识"。

## 关键术语速查

| 术语 | 中文 | 简单解释 |
|------|------|----------|
| Algorithm | 算法 | 解决问题的步骤和方法 |
| Model | 模型 | 训练完成后用于预测/生成的"知识体" |
| Training | 训练 | 让模型从数据中学习的过程 |
| Inference | 推理 | 用训练好的模型做预测的过程 |
| Dataset | 数据集 | 用于训练和测试的数据集合 |
| Feature | 特征 | 用于描述数据的属性（如身高、体重） |
| Label | 标签 | 数据的正确答案（如"猫"、"狗"） |
| Parameter | 参数 | 模型内部的可调数值，决定模型行为 |
| Overfitting | 过拟合 | 模型只记住了训练数据，对新数据表现差 |

## ✅ 本章小结

- **AI** 是让机器模拟人类智能的学科
- **机器学习**是 AI 的核心方法，**深度学习**是 ML 的重要分支
- AI 经历了从诞生→寒冬→复兴→爆发的曲折历程
- 当前所有 AI 都属于 **狭义 AI**，通用 AI 仍在路上
- AI 已经广泛应用于医疗、交通、金融、创意等各个领域
