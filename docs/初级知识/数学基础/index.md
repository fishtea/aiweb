# 数学基础

学习 AI 不需要一开始就掌握高等数学全貌，但需要理解几个核心数学工具。它们帮助你看懂模型如何表示数据、如何学习、如何判断好坏。

## 最小必要数学

| 模块 | 为什么重要 | 入门重点 |
|------|------------|----------|
| 线性代数 | AI 把文本、图片、用户行为表示成向量和矩阵 | 向量、矩阵乘法、维度、点积 |
| 概率统计 | 模型输出通常是概率，评估也依赖统计 | 均值、方差、概率分布、条件概率 |
| 微积分 | 神经网络通过梯度下降优化参数 | 导数、梯度、链式法则 |
| 优化方法 | 训练模型就是最小化损失函数 | 损失函数、学习率、局部最优 |
| 信息论 | 分类、语言模型和压缩都涉及不确定性 | 熵、交叉熵、困惑度 |

## 向量和矩阵

AI 系统通常不会直接处理“文字”或“图片”，而是先把它们变成数字。

```text
一条用户评论 → 文本向量
一张图片 → 像素矩阵或视觉特征向量
一个用户 → 年龄、城市、行为次数等特征向量
```

向量可以理解为一组有顺序的数字。两个向量越相似，通常表示它们代表的对象越接近。Embedding、推荐系统、向量数据库都建立在这个思想上。

## 概率思维

模型很少给出绝对确定的答案。分类模型常输出每个类别的概率：

| 类别 | 概率 |
|------|------|
| 正面 | 0.82 |
| 中性 | 0.13 |
| 负面 | 0.05 |

概率最高的类别不一定永远正确，它只代表模型在当前输入下的置信程度。实际系统还需要阈值、人工复核和错误处理。

## 损失函数

损失函数用来衡量模型预测和真实答案之间的差距。训练过程的目标是让损失越来越小。

| 任务 | 常见损失 | 直觉 |
|------|----------|------|
| 回归 | 均方误差 MSE | 预测值离真实值越远，惩罚越大 |
| 分类 | 交叉熵 | 正确类别概率越低，惩罚越大 |
| 语言模型 | 下一个 token 预测损失 | 预测真实下一个 token 的能力 |

## 梯度下降

梯度下降可以理解为“沿着损失下降最快的方向调整参数”。

```text
初始化参数 → 计算预测 → 计算损失 → 计算梯度 → 更新参数 → 重复
```

学习率决定每次更新走多远。学习率太大，训练可能震荡；学习率太小，训练会非常慢。

## 初学者常见误区

- 不必先把所有公式推导完再学应用，先理解直觉和用途。
- 维度比公式更重要，很多代码错误来自矩阵维度不匹配。
- 概率不是保证，模型输出 90% 仍可能出错。
- 训练损失下降不代表模型在真实场景中一定更好。

## 学习建议

先用图形和代码理解概念，再补公式。推荐顺序是：

1. 向量、矩阵、点积、余弦相似度
2. 均值、方差、正态分布、条件概率
3. 导数、偏导数、梯度
4. 损失函数、梯度下降、学习率
5. 交叉熵、softmax、困惑度
